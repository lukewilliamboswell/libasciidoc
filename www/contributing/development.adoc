== Prerequisites

* https://go.dev/dl/[Go] 1.21 or later
* Git

== Clone and Build

 git clone https://github.com/lukewilliamboswell/libasciidoc.git
 cd libasciidoc
 go build ./...

Build the CLI tools:

 go build -o ascii2html ./cmd/ascii2html
 go build -o ascii2doc ./cmd/ascii2doc

== Run Tests

 go test ./...

== Run the Site Locally

To preview documentation changes:

 ./serve.sh

This builds `ascii2doc`, generates the site into `_site/`, and serves it on `http://localhost:8080`.
