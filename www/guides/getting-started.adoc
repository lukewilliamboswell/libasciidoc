= Getting Started
:weight: 1

== Installation

Install with Go:

 go install github.com/lukewilliamboswell/libasciidoc/cmd/ascii2doc@latest
 go install github.com/lukewilliamboswell/libasciidoc/cmd/ascii2html@latest

== Basic Usage

Convert a single file to HTML:

 ascii2html myfile.adoc

Convert a single file to DOCX:

 ascii2doc myfile.adoc

== Static Site

Build a static site from a directory of `.adoc` files:

 ascii2doc --static-site -o _site/ www/
